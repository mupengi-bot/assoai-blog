AssoAI의 핵심은 **AI 에이전트**다.

단순 챗봇이 아니라, 실제로 일을 처리하는 자동화 시스템.

## AI 에이전트란?

기존 챗봇: 질문 → 답변 (끝)

AI 에이전트: 질문 → 이해 → **실행** → 결과

차이점? **실제로 행동한다.**

## AssoAI 에이전트 종류

### 1. Meeting Prep Agent
회의 전 자동 준비
- 참석자 일정 확인
- 관련 문서 수집
- 안건 정리

### 2. Event Reminder Agent
행사 리마인더 자동화
- D-7, D-3, D-1 알림
- 준비물 체크리스트
- 담당자별 태스크 분배

### 3. Inbox Processor Agent
받은 메시지 자동 처리
- 중요도 분류
- 자동 태깅
- 우선순위 정렬

### 4. Task Tracker Agent
업무 진행 상황 추적
- 마감일 알림
- 진행률 시각화
- 병목 지점 알림

### 5. Organization Analyzer Agent
조직 현황 분석
- 활동 패턴 분석
- 참여도 측정
- 개선점 제안

## 어떻게 동작하나?

```
트리거 (시간/이벤트/수동)
    ↓
에이전트 실행
    ↓
외부 서비스 연동 (Calendar, Slack, Notion...)
    ↓
결과 알림
```

## 프라이버시

- 모든 데이터는 암호화 저장
- 외부 공유 없음
- 사용자 동의 기반 접근
